# growcat
아주 귀여운 고양이판 짭미니게임천국

실행하는법
---
1. "'Node.js"'로 살행합니다
2. Node.js prompt에서 npm으로 "'express"'와 "'socket.io"'를 설치합니다
3. Node.js에서 server를 실행합니다(http://localhost:8081)
